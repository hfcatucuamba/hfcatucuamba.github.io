JavaScript HTML DOM - Cambio de HTML

CONCEPTO

El HTML DOM permite que JavaScript cambie el contenido de los elementos HTML.

Cambio de contenido HTML

Para hacer este proceso usamos la propiedad de innerHTML que nos permite cambiar el contenido de un elemento.

Para cambiar el contenido de un elemento HTML, utilice esta sintaxis:

  • document.getElementById(id).innerHTML = new HTML


Ejemplo

Este ejemplo cambia el contenido de un <p> elemento:

Código

<!DOCTYPE html>
<html>
<body>
<h2>JavaScript puede cambiar HTML</h2>
<p id="p1">Hello World!</p>
<script>document.getElementById("p1").innerHTML = "New text!";</script>
<p>El párrafo anterior fue cambiado por un script.</p>
</body>
</html>

Ejecución

JavaScript puede cambiar HTML

Hello World!

El párrafo anterior fue cambiado por un script.

Click en el botón para cambiar el contenido <p> por Hola Mundo!



Cambiar el valor de un atributo

Para cambiar el valor de un atributo HTML, use esta sintaxis: document.getElementById(id).attribute = nuevo valor

Este ejemplo cambia el valor del atributo src a un elemento: <img>

Código

<!DOCTYPE html>
<html>
<body>
<h2>JavaScript HTML DOM</h2>
<img id="image" scr="../img/car.jpg"></p>
<script>document.getElementById("image").src = "../img/programacion.gif";</script>
<p>La imagen original era un auto.jpg, pero se la cambio por un gif.</p>
</body>
</html>

Ejecución

JavaScript HTML DOM

La imagen original era un auto.jpg, pero se la cambio por un gif.

Click en el botón para cambiar la imagen



Contenido HTML dinámico

JavaScript puede crear contenido HTML dinámico: Fecha: Dom 04 Dic 2022 15:33:45 GMT-0500 (hora de Ecuador)

Código

<!DOCTYPE html>
<html>
<body>
<p id="demo"></p>
<script>document.getElementById("demo").innerHTML = "Date : " + Date();</script>
</body>
</html>

Ejecución

Click en el botón para ver la fecha actual
