Validación de formulario JavaScript

La validación de formularios HTML se puede realizar mediante JavaScript.
Si un campo de formulario (fname) está vacío, esta función alerta con un mensaje y devuelve falso para evitar que se envíe el formulario:

Código

<!DOCTYPE html>
<html>
<head>
<script>function validateForm(){
let x = document.forms["myForm"]["fname"].value;
if (x == "") {
alert("Se debe completar el nombre");
return false;
} } </script>
</head>
<body>
<h2>Validación de JavaScript</h2>
<form name="myForm" onsubmit="return validateForm()" method="post"></p>
Nombre:<input type="text" name="fname"></p>
<input type="submit" value = "Submit"></p>
</body>
</html>

Ejecución

Click en el botón para validar el formulario.



JavaScript puede validar la entrada numérica

JavaScript se usa a menudo para validar la entrada numérica:

Código

<!DOCTYPE html>
<html>
<body>
<h2>Validación de JavaScript</h2>
<p>Por favor ingrese un número entre 1 y 10:</p>
<input id="numb">
<button type="button" onclick="myFunction()">Enviar</p>
<p id="demo"></p>
<script>function myFunction(){
let x = document.getElementById("numb").value;
let text;
if (isNaN(x) || x < 1 || x > 10) {
text = "Entrada no válida";
return false;
}else{
text = "Entrada correcta";
}
document.getElementById("demo").style.color = 'red'
document.getElementById("demo").innerHTML = text;
} </script>
</body>
</html>
