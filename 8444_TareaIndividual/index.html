AJAX

Que es AJAX?

AJAX es el acrónimo de Asynchronous JavaScript and XML (JavaScript asíncrono y XML). Es un grupo de tecnologías interrelacionadas como JavaScript, DOM, XML, HTML/XHTML, CSS, XMLHttpRequest, etc. Nos permite enviar y recibir datos de forma asíncrona sin recargar la página web. Por lo tanto, es rápido.

XML

¿Por qué estudiar XML?

XML juega un papel importante en muchos sistemas de TI diferentes. XML se utiliza a menudo para distribuir datos a través de Internet. Es importante (¡para todos los tipos de desarrolladores de software!) tener una buena comprensión de XML.

JSON

¿Por qué estudiar JSON?

El formato JSON es sintácticamente similar al código para crear objetos JavaScript. Debido a esto, un programa JavaScript puede convertir fácilmente datos JSON en objetos JavaScript.Dado que el formato es solo texto, los datos JSON se pueden enviar fácilmente entre computadoras y ser utilizados por cualquier lenguaje de programación.

Aprendamos Mas !!

DIFERENCIAS XML Y JSON

JSON XML
JSON se amplía desde JavaScript XML se ha ampliado desde SGML
JSON admite tipos de datos y matrices XML no proporciona tipos de datos y no admite matrices.
JSON no admite Comentarios Soporta XML para Comentarios
JSON es mejor para los servicios web XML es mejor para la configuración.
Los datos JSON no se pueden cambiar a otros formatos XML se pueden cambiar a otros formatos como texto sin formato
JSON está orientado a datos XML está orientado a documentos.

SEMEJANZAS XML Y JASON

XML JASON
XML son formatos autodescriptivos JSON son formatos autodescriptivos
XML un buen soporte para las definiciones de contenido y la validación JSON un buen soporte para las definiciones de contenido y la validación
XML pueden utilizarse para la serialización de datos (convertir una estructura de datos a un formato para su almacenamiento o transmisión). JSON pueden utilizarse para la serialización de datos
XML están basadas en texto Jason están basadas en texto
XML tienen estructuras jerárquica Jason tienen estructuras jerárquica
