Introducción a AJAX - JQuery

Concepto

AJAX significa "JavaScript asíncrono y XML". JavaScript incluye funciones de envío de solicitudes http asincrónicas utilizando XMLHttpRequestun objeto. Ajax se trata de usar esta capacidad de JavaScript para enviar una solicitud http asíncrona y obtener los datos xml como respuesta (también en otros formatos) y actualizar parte de una página web (usando JavaScript) sin recargar o actualizar toda la página web.



La figura anterior ilustra la funcionalidad de Ajax.

La biblioteca jQuery incluye varios métodos para enviar solicitudes Ajax. Estos métodos utilizan internamente el objeto XMLHttpRequest de JavaScript. La siguiente tabla enumera todos los métodos Ajax de jQuery.


Métodos jQuery Ajax

Método Descripción
ajax() Envía una petición http asíncrona al servidor.
get() Envía una petición http GET para cargar los datos desde el servidor.
Post() Envía una petición http POST para enviar o cargar los datos al servidor.
getJSON() Envía una petición http GET para cargar datos codificados en JSON desde el servidor.
getScript() Envía una petición http GET para cargar el archivo JavaScript desde el servidor y luego lo ejecuta.
load() Envía una petición http para cargar el contenido html o de texto desde el servidor y añadirlo a los elementos DOM.

Eventos

La biblioteca jQuery también incluye los siguientes eventos que se activarán según el estado de la solicitud de Ajax.

jQuery Ajax Eventos Descripción
ajaxComplete() Registre una función de controlador para que se llame cuando se completen las solicitudes de Ajax.
ajaxError() Registre una función de controlador para que se llame cuando las solicitudes de Ajax se completen con un error.
ajaxSend() Registre una función de controlador para que se llame antes de que se envíe la solicitud de Ajax.
ajaxStart() Registre una función de controlador para que se llame cuando comience la primera solicitud de Ajax.
ajaxStop() Registre una función de controlador para que se llame cuando se hayan completado todas las solicitudes de Ajax.
ajaxSuccess() Registre una función de controlador para que se llame cuando la solicitud de Ajax se complete correctamente.

Ventajas de jQuery Ajax

  • Compatibilidad con varios navegadores
  • Métodos simples de usar
  • Capacidad para enviar solicitudes GET y POST
  • Capacidad para cargar JSON, XML, HTML o scripts
  • Reduzca el tráfico del servidor y aumente la velocidad
  • Validación de formulario
  • Habilitar llamadas asíncronas
  • Gran apoyo de la comunidad
  • Fácil de usar con otras funciones de jQuery
  • Soporta una variedad de tipos de datos

Desventajas de jQuery Ajax

  • Fuente abierta.
  • Dependencia de la biblioteca jQuery
  • Mayor tamaño de archivo
  • Rendimiento potencialmente más lento
  • Problemas de seguridad
  • Problemas de compatibilidad
  • Dependencia de una biblioteca de terceros
  • Control limitado
  • Biblioteca obsoleta
