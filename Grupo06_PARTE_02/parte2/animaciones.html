Estilo de los Elementos

El elemento contenedor debe crearse con estilo = " position: relative".
El elemento de animación debe crearse con estilo = " position: absolute".

Código

<!DOCTYPE html>
<html>
<body>
<h4>Animación de JavaScript</h4>
<style>
#container {
 width: 400px;
 height: 400px;
 position: relative;
 background: whitesmoke;
}
#animate, #img{
 width: 100px;
 height: 100px;
 position: absolute;
}
</style>
<div id ="container">
 <div id ="animate"><img src="assets/img/camino.png" id="img"></div>
</div>
</body>
</html>

Código de Animación

Las animaciones de JavaScript se realizan mediante la programación de cambios graduales en el estilo de un elemento. Los cambios son llamados por un temporizador. Cuando el intervalo del temporizador es pequeño, la animación parece continua.

Código

<!DOCTYPE html>
<html>
<body>
<h4>Animación de JavaScript</h4>
<style>
#container {
 width: 400px;
 height: 400px;
 position: relative;
 background: whitesmoke;
}
#animate, #img{
 width: 100px;
 height: 100px;
 position: absolute;
}
</style>
<div id ="container">
 <div id ="animate"><img src="assets/img/camino.png" id="img"></div>
</div>
<p><button onclick="myMove()">Pruébalo</button></p>
<script>function myMove() {
 let id = null;
 const elem = document.getElementById("animate");
 let pos = 0;
 clearInterval(id);
 id = setInterval(frame, 5);
 function frame() {
  if (pos == 300) {
  clearInterval(id);
 } else {
  pos++;
  elem.style.top = pos + "px";
  elem.style.left = pos + "px";
 }
}
}
</script>
</body>
</html>
