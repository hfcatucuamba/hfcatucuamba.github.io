El objeto HTMLCollection

El getElementsByTagName()método devuelve un HTMLCollectionobjeto.
Un HTMLCollectionobjeto es una lista similar a una matriz (colección) de elementos HTML.
El siguiente código selecciona todos los <p> elementos en un documento:
Se puede acceder a los elementos de la colección mediante un número de índice, el índice comienza en 0.

Código

<!DOCTYPE html>
<html>
<body>
<h2 class="titulo">Ejecucion</h2>
<h2>JavaScript HTML DOM Colecciones</h2>
<p>Hola Mundo!</p>
<p>Hola Ecuador!</p>
<p id="demo"></p>
<script>
 const myCollection = document.getElementsByTagName("p");
 document.getElementById("demo").innerHTML = "El innerHTML del segundo párrafo es:" + myCollection[1].innerHTML;
</script>

</body>
</html>

HTMLCollection Longitud

La propiedad length define el número de elementos en un HTMLCollection y es útil cuando desea recorrer los elementos de una colección:

Codigo

<!DOCTYPE html>
<html>
<body>
<h2>JavaScript HTML DOM Colecciones</h2>
<p>Hola Mundo!</p>
<p>Hola Ecuador!</p>
<p>Haga clic en el botón para cambiar el color de todos los elementos p.</p>
<button onclick="myFunction1()">Pruebalo</button>
<script>
 function myFunction1() {
  const myCollection = document.getElementsByTagName("p");
  for (let i = 0; i < myCollection.length; i++) {
   myCollection[i].style.color = "green";
  }
 }
</script>

</body>
</html>
